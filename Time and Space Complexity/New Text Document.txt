Time Complexity counting with Examples:-
Time complexity প্রকাশ করা হয় মুলত ৩ টা Asymptotic Notation এর মাধ্যমে। এগুলো হলঃ

Big-O Notation (O-notation)
— It provides the worst-case complexity of the algorithm.

Omega Notation (Ω-notation) —
— It provides the best-case complexity of an algorithm.

Theta Notation (Θ-notation)
— It provides the average-case complexity of an algorithm.

উপরের Notation গুলোর মধ্যে থেকে আমরা Big-O Notation (O-notation) দিয়েই worst-case Complexity calculate করবো।

2. Arithmatic Operation —

Program এ কোনো arithmatic operation (যোগ, বিয়োগ, গুণ, ভাগ) হলে তখন TC calculate করা হয়। উপরে var c = a+b, তে arithmatic operation চলবে, এই সময় TC calculate করা হবে।

এই দুটো অপারেশন কতবার একটা program এ হইছে সেটা count করা হয়।
